Astral Classification using Spectral Characteristics

In astronomy, Astral classification refers to the classification of stars based on their spectral characteristics. This project aims to build a Machine Learning model that predicts the class of a stellar object (star, galaxy, or quasar) based on space observation data from the Sloan Digital Sky Survey (SDSS).

Key Features:

  • Alpha (Right Ascension angle)
  • Delta (Declination angle)
  • Ultraviolet, Green, Red, Near Infrared, and Infrared filters
  • Run ID, Rerun ID, Camera column, and Field ID
  • Redshift value, Plate ID, MJD, and Fiber ID
  • Class label (Star, Galaxy, or Quasar)

Naive Bayes Algorithm

The Naive Bayes algorithm is a simple yet powerful classification algorithm based on Bayes' Theorem. It assumes independence between predictors and is particularly effective for large datasets with categorical inputs. Despite its simplicity, Naive Bayes performs surprisingly well in various real-world applications like spam detection, sentiment analysis, and document classification.

Vegetable Classifier

This project involves building a machine learning model to classify vegetables into 15 distinct categories, such as Bean, Bitter Gourd, Brinjal, Broccoli, and Tomato. Using image data, the model learns to identify key visual features that distinguish each vegetable, making it useful for applications like smart farming and automated grocery systems.
